### LLM assistant — March 2024 - Current — [Source](https://github.com/madacol/whatsapp-llm-bot)

An LLM agent that can receive messages from different sources (whatsapp, discord, etc) and can run JavaScript code in an environment with access to tools and a DB, and can also create/edit tools on-the-fly

**Tech Stack:** NodeJs, LLM API, PGlite (PostgreSQL in WASM)\
**Hosted on:** VPS
