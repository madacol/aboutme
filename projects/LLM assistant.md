### LLM assistant — March 2024 - Current — [Source](https://github.com/madacol/whatsapp-llm-bot)

An Agents' harness that can connect multiple transports (like whatsapp) and let any ACP-compatible agent (like Claude code, Codex, or Pi via ACP adapters) handle the response and processing. This is very similar to OpenClaw or Hermes Agent, but with a UI closer to CLI agents, with a focus for developing and reviewing. It's possibly the best way to use Codex via Whatsapp

**Tech Stack:** NodeJs, LLM API, PGlite (PostgreSQL in WASM)\
**Hosted on:** VPS
