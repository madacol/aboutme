### LLM assistant — March 2024 - Current — [Source](https://github.com/madacol/whatsapp-llm-bot)

Use Codex/Claude code via Whatsapp. A harness that can connect multiple transports (like whatsapp) and let any ACP-compatible agent (like Claude code, Codex, or Pi via ACP adapters) handle the response. Similar to OpenClaw or Hermes Agent, but less features and focused for developing and reviewing, with almost as much details as the CLI agents

**Tech Stack:** NodeJs, LLM Agents (Codex / Claude / Pi), Sqlite
